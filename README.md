# ❌⭕ Tic Tac Toe Game

A dynamic and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This version supports a customizable board (currently 4×4) and includes win/draw detection with a smooth user experience.

---

## 🚀 Features

- 🎮 Two-player mode (X & O)  
- 🔢 4×4 game board (customizable size)  
- 🧠 Win detection:
  - Rows  
  - Columns  
  - Diagonals  
  - Reverse diagonals  
- 🤝 Draw detection  
- 🔄 Reset button to restart the game  
- ⌨️ Keyboard support (Press "Enter" to play)  
- 🎨 Dynamic UI updates for each move  

---

## ▶️ How to Play

1. Open the game in your browser  
2. Player **X** starts first  
3. Click on a cell (or press Enter when focused)  
4. Players take turns  
5. First player to complete a row, column, or diagonal wins 🎉  
6. If all cells are filled → Draw 🤝  
7. Click **Reset** to play again
